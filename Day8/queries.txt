--------------------------SQL CRUD Operations (Hands-On)--------------------------------------

## Step 1: Create a Table

- SQL QUERY: 

CREATE TABLE students (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  email TEXT UNIQUE NOT NULL,
  age INTEGER,
  course TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

- OUTPUT : CREATE TABLE
